# Secure-MultiSite-Network-MPLS-IPsec
Simulation d'un réseau FAI sécurisé pour interconnecter des sites distants. Isolation totale des clients via MPLS L3VPN (VRF/BGP) et chiffrement des échanges critiques par tunnels IPsec. Projet réalisé sur GNS3 incluant la configuration du backbone, du routage dynamique et des politiques de sécurité (ACL, QoS)
